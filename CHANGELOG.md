# Change Log

All notable changes to the "Java Method Sorter" extension will be documented in this file.

## [1.0.0] - 2026-01-10

### Added
- Initial release
- Sort methods in Java classes by access level, invocation order, and lexical order
- Shuffle methods randomly feature
- Configuration options for sorting strategy
- Context menu integration under tlcsdm submenu
- Support for clustering overloaded methods
- Support for clustering getter/setter methods
- Keyboard shortcut (Alt+S) for sort methods command
